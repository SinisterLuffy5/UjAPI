## Features
1. Custom Launcher.
2. Multi-Window.
3. Lua support.
4. New custom natives.
5. Multi-patch support.

## How to use

1. Download and Extract to a separate folder.
2. Launch UjAPI.exe, configure path to Warcraft 3 and World Editor.
3. Use "Launch" button for Warcraft 3 or World Editor.
4. Enjoy.
